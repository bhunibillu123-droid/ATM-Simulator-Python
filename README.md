# 🏦 ATM Simulator Using Python

A simple and user-friendly ATM Simulator developed using Python. This project demonstrates fundamental Python programming concepts such as conditional statements, loops, lists, user input handling, and menu-driven programming. It simulates the basic functionalities of an Automated Teller Machine (ATM) in a console-based application.

---

## 📌 Features

- 🔐 Secure PIN Authentication
- 💰 Check Account Balance
- 💵 Deposit Money
- 💸 Withdraw Money
- 📄 View Mini Statement
- 🔄 Change ATM PIN
- 🚪 Exit the Application
